Introduccion a CSS

Selectores

Los selectores haceb referencia a que etiqueta se le va a dar estilo , los selectores son las etiquetas del HTML por ejm el h1 , p , h2, etc

Existen tipos de selectores

Selector de etiqueta

Este selector hace referencia a la etiqueta HTML

En este ejemplo el daremos al h3 un color rojo

Selector de clase

El selector de clase es el que mas utilizaremos en dar estilos a las etiquetas , es como un identificador a la etiqueta que se le va a dar estilo , algo asi como un apodo o sobrenombre y en el archivo css se manda a llamar con un punto y el nombre de la clase .nombreClase , pueden haber varias etiquetas con el mismo nombre de clase

En este ejemplo le daremos un colo azul

Selector de ID

El selector Id es un identificador unico en las etiquetas osea si tu le pones un nombre de id a un etiqueta , ese nombre no debe de ser llamado en otra etiqueta , mejor dicho no se puede mandar a llamar dos id con el mismo nombre, se manda a llamar con el michi #nombreId

En este ejemplo le daremos el color de peru

Selector inline

Este selector es el que tiene como atributo llamado style y le pones los estilos

En este ejemplo le daremos un colo salmon

Herencia

La herencia es como decir el padre tiene un estilo pues los hijos lo heredan , si un div por ejemplo tiene hijos como un h1 y un parrafo y a ese div le pones color rojo , los hijos heredan ese coloc, ahora hay casos en lo que no se heredan , para eso puedes utilizar la el valor inherit en el heredero osea el hijo

Cascada

Cascada es como su nombre lo dice pues de arriba hacia abajo y el que esta en lo ultimo es el que toma el valor con mas fuerza o el que se le aplica

Especificidad

La escpecificidad son valores ,es como un numero con una cantidad de fuerza segun los tipos de selctores , por ejemplo el menos fuerte es el selector de etiqueta y el mas fuerte es el !important , pero como no es recomendable usar quedaria el #id la cual en estilos no se recomienda usar , mas se usaria con JS
